URL Shortener Microservice

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON respons
  • When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

https://minute-url.herokuapp.com/new/https://www.google.com

Example creation output

{ "original_url":"http://foo.com:80", "short_url":"https://minute-url.herokuapp.com/4682" }

Usage:

https://minute-url.herokuapp.com/4682

Will redirect to:

https://www.google.com/
